# CodeAlpha_URLShortener

A simple URL Shortener built using Node.js, Express.js, and SQLite.

## Features

* Generate unique short URLs
* Redirect short URLs to original URLs
* SQLite database storage
* URL validation
* Click count tracking
* URL history table
* Simple and clean user interface

## Installation

1. Install dependencies

```bash
npm install
```

2. Start the server

```bash
npm run dev
```

3. Open in browser

```
http://localhost:3000
```
